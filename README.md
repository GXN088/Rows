# Rows
Number Pattern



Челлендж

Легко
Каждый тест-кейс имеет один ввод — нечетное целое число.
Ваша задача — вывести n-пирамиду используя *, вот некоторые примеры:

1 - пирамида
*
5 - пирамида
*
***
*****
7 - пирамида
*
***
*****
*******
Ввод
нечетное целое число n от пользователя
1 <= n < 1000
Советы
Попробуйте начать с маленьких треугольников
Проверьте подсказку, если застряли ;)
n представляет количество * в нижнем ряду

Обратите внимание, что для пирамиды размером n у вас будет (n / 2) + 1 строк, поэтому:

int n = scanner.nextInt();
int rows = n / 2 + 1;
for (int i = 0; i < rows; i++) {
    ...

Каждая итерация цикла выводит часть пирамиды сверху вниз, используйте i и n


┌─────────────┐
│   start     │
└──────┬──────┘
       ↓
┌─────────────┐
│  input n    │
└──────┬──────┘
       ↓
┌─────────────┐
│ rows=n/2+1  │
└──────┬──────┘
       ↓
┌─────────────┐
│   i = 0     │
└──────┬──────┘
       ↓
  ┌────┴────┐
  │ i<rows? │──НЕТ──→ КОНЕЦ
  └────┬────┘
       │ДА
       ↓
┌─────────────┐
│ stars=2*i+1 │
└──────┬──────┘
       ↓
┌─────────────┐
│   j = 0     │
└──────┬──────┘
       ↓
  ┌────┴─────┐
  │ j<stars? │──НЕТ──→
  └────┬─────┘        │
       │ДА            ↓
       ↓        ┌─────────────┐
┌─────────────┐ │ println()   │
│ print("*")  │ │ (перевод    │
└──────┬──────┘ │  строки)    │
       ↓        └──────┬──────┘
┌─────────────┐        │
│ j = j + 1   │        ↓
└──────┬──────┘   ┌─────────────┐
       └─────────→│ i = i + 1   │
                  └──────┬──────┘
                         ↓
                       (наверх к проверке i<rows)

                       НАЧАЛО
    ВВЕСТИ n (нечётное число)
    
    rows = n / 2 + 1        // количество строк в пирамиде
    
    ДЛЯ i ОТ 0 ДО rows - 1:
        stars = 2 * i + 1   // сколько звёздочек в текущей строке
        
        ДЛЯ j ОТ 0 ДО stars - 1:
            ВЫВЕСТИ "*"     // без перевода строки
        КОНЕЦ ЦИКЛА j
        
        ВЫВЕСТИ ПЕРЕВОД СТРОКИ  // переходим на следующую строку
    КОНЕЦ ЦИКЛА i
    
КОНЕЦ
